Sobre el Proyecto

Este proyecto de robótica usa sensores y actuadores para investigar si la experiencia tecnológica puede ser percibida como placer o felicidad. El robot está construido con componentes electrónicos accesibles y un diseño pensado para aplicaciones prácticas y educativas.

Objetivos Generales

  • Desarrollar un robot que responda a estímulos ambientales.
  • Demostrar la integración entre hardware y software.
  • Crear una experiencia interactiva que conecte emoción y tecnología.

Objetivos Específicos

  • Implementar sensores de movimiento, luz y proximidad.
  • Construir un sistema de control con microcontrolador.
  • Registrar datos y mantener el robot seguro y funcional.

Funcionamiento

El robot funciona mediante ciclos de sensor-actuador donde detecta el entorno, procesa la información y responde con movimientos o señales luminosas.

  1. Captura: Los sensores detectan luz, distancia y movimiento.
  2. Procesamiento: El microcontrolador evalúa los datos y decide la acción.
  3. Actuación: El robot mueve motores, enciende LEDs o envía alertas.
  4. Retroalimentación: El sistema ajusta su comportamiento según la respuesta.

Mantenimiento Preventivo

Mantén el robot operativo con buenas prácticas de limpieza y revisión periódica.

Limpieza

Quita polvo con aire comprimido y limpia contactos con alcohol isopropílico.

Verificación

Revisa conexiones, tornillos y soldaduras para evitar fallos.

Actualización

Actualiza firmware y el software de control cuando sea necesario.

Almacenaje

Guarda el robot en un lugar seco y sin humedad para proteger sus componentes.

Normas de Seguridad

ESD

Usa pulsera antiestática y mantén componentes sensibles en superficies protegidas.

Herramientas

Utiliza destornilladores, alicates y soldadores adecuados y en buen estado.

Alimentación

Desconecta la alimentación antes de manipular circuitos y evita cortocircuitos.

Protección

Usa gafas y guantes siempre que trabajes con componentes eléctricos o mecánicos.
